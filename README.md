# MathApp
## О чем наше приложение:  
Мобильное приложение малого матфака для контроля посещаемости учеников, возможностью просмотра и изменения расписания и дополнительной информации, необходимой для учебного процесса

## Stack
- Kotlin
- Compose
- Retrofit + OkHttp
- Dagger + Hilt
- MVVM
- Clean Architecture

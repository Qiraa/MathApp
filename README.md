## О чем наше приложение:  
Мобильное приложение малого матфака для контроля посещаемости учеников, возможностью просмотра и изменения расписания, а также дополнительной информации, необходимой для учебного процесса

## Стек
- Kotlin
- Compose
- Retrofit + OkHttp
- Koin
- MVVM
- Clean Architecture

## Ограничения
- Версия Android 7+
- На данный момент не поддерживает темную тему, также нет определенной темы, но есть поддержка Dynamic Colors (Android 12+)
- Регистрация только через администрацию Малого Матфака
